1. ano ang kahalagahan ng buwan ng wika


Para mapalaganap ang yaman na meron tayong mga Pilipino.

Mahalaga ang buwan ng wika dahil dito ay naipapaalala sa mga bagong henerasyon ngayon ang wika ng pilipino.Sapagkat karamihan ngayon ay mas tinatangkilik nating mga pilipino ang ibang wika na nagiging dahilan upang unti-unting mawala ang diwa ng ating wika.Sa tulong ng Buwan ng wika na inilulunsad sa mga paaralan.Ay naibabalik ng kaunti ang diwa ng wikang pambangsang tagalog.


2. Ano ang kahalagahan ng pagdiriwang ng buwan ng wika?​


Answer:

Ang kahalagahan ng pagdiriwang ng buwan ng wika ay mahalaga dahil ito ay ang bahagi ng ating kasaysayan, ito rin ang nagpapanatili sa ating pagkakaisa bilang isang bansa. Ipinapakita ng wika ng isang bansa kung gaano mayaman at natatangi ang kultura at pamana nito. Higit pa sa pagiging isang paraan para sa komunikasyon, ang mga tulay ng wika sa iba't ibang mga pamayanan at pinalalakas ito.
