23. bakit ang wika ay midyum ng pakikipagtalastasan? ​


Ang wika ang midyum ng pakikipagtalastasan sapagkat eto ang pinakamabisang paraan upang masabi natin ang ating mga niloloob. Sa pamamagitan ng wika nasasabi natin sa iba ang ating mga ideya at kaisipan. Maaari natin ma-iparating ang ating saloobin sa pamamagitan ng gesture, pagtingin o iba pa. Ngunit ang wika at pagsasalita ang pinaka malinaw na paraan upang masabi ang ating nais sabihin sa iba. Isa pa ang wika ay may ibat-ibang uri at katangian.  

Mga Katangian Ng wika  

Ang mga sumusunod ay ang mga katangian ng wika:  

Ang wika ay sistematiko balangkas  Ang wika ay nabubuo sa pamamagitan ng mga tunog  Ang wika ay Arbitraryo  Ang wika ay Dinamiko    Ang wika ay nagbabago  Ang wika ay nakadepende sa kultura  

Uri Ng Wika  

Ang mga sumusunod ay ang mga uri ng wika:  

Wikang pambansa – tulad ng japanese, lao, Frisian, German,    Wikang balbal- Tulad ng Beshie, petmalu, praning, Wikang Pampanitikan- Mga wikang ginagamit sa pag-gawa o pagkatha ng mga tula, novela o maikling kento  Wikang panlalawigan- Tulad ng ilocano, cebuano, Tausug, maranao
