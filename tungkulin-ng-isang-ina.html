3. ano ang tungkulin ng isang ina


tungkulin ng isang ina na pangalagaan ang kaniyang pamilya, mahalin at ibigay ang mga pangangailangan nito sa araw araw.

Ang papel ng isang ina ay mahalin ang kanyang mga anak nang buong puso niya. Ito rin ang papel ng bawat ina na maunawaan ang kanyang mga anak. ... Ang mga ina ay kailangang mag-ehersisyo upang maunawaan, tulungan at mahalin ang bata. Ang isa sa mga pinakamahalagang papel na ginagampanan ng ina ay ang papel ng tagapangalaga.

13. Ipaliwanag ang tungkulin ng isang ina. _________________________​


Answer:

Ang tungkulin ng isang ina ay ang pangalagaan ang tahanan. Tungkulin ng isang ina siguruduhin ang kalinisan ng tahanan. Tungkulin ng isang ina na siguruduhin na nasa maayos na kalagayan ang kaniyang mga anak. Tungkulin ng isang ina huwag pabayaan ang kalusugan ng kanyang pamilya. Tungkulin ng isang ina na mahalin at alagaan ang sarili niya.

21. bakit mahalaga ang pagtupad ng mga tungkulin ng isang ina at anak?


Mahalaga ang pagtupad ng mga tungkulin ng isang ina at anak dahil ito ang nagiging paraan upang maging mabuti ang komunikasyon o pamumuhay ng isang pamilya.Isa pa ay nagiging mabuting paran din ito upang maging masaya ang mga pamilya.


22. sa isang pamilya, ano ang iba't ibang tungkulin ng isang ina at ama


Ang ina ay ang ilaw ng tahanan.Sila ang gumagabay o nagbibigay ng direksyon sa lahat ng miyembro ng pamilya. Nagpapa liwanag sa tahanan sa pamamagitan ng pag-aalaga, pagpapakita ng pag-aaruga at pagmamahal sa bawat miyembro ng pamilya.Sa bawat pamilya ay may kinikilalang ina. Ina na kailanman ay nagbibigay ng liwanag sa buong sulok ng tahanan.

Bilang haligi ng tahanan, inaasahan na itataguyod niya ang kaniyang pamilya at sisiguruhin ang kaligtasan ng bawat isa.Hindi lang ito dahil may kakayahan din ang nanay na maghanap-buhay, kundi dahil nakakabuti para sa bata ang pag-aalaga at presensya ng isang ama.Sa kaniya rin nakapasan ang malaking responsibilidad ng paghahanap-buhay para sa pangangailangan ng kaniyang mga anak.
